# ticketing-practice
공연 티켓 예매 연습 웹 애플리케이션

- 예매가 아직 열리지 않은 공연들을 메인 페이지에 보여준다.
- 예매 날짜가 가까운 공연부터 우선 순위로 보여준다.
- 장소가 같으면 모두 동일한 트래픽을 처리한다.
- 관리자는 개발자 혼자이고, 회원가입을 새로 하는 모든 사람들은 일반 사용자이다.
